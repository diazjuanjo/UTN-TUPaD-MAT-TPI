## **2\. DESCRIPCIÓN DEL PROBLEMA**

### **Fundamentación Teórica**

El presente proyecto tiene como objetivo vincular los conceptos abstractos del **Álgebra de Boole** con su aplicación práctica en la programación. El Álgebra de Boole es una estructura matemática que esquematiza las operaciones lógicas, trabajando con un conjunto binario: **0 (Falso/OFF)** y **1 (Verdadero/ON)**.

El programa desarrollado ("Simulador de Compuertas Lógicas") permite al usuario visualizar y calcular el resultado de las operaciones fundamentales que rigen la lógica digital de las computadoras:

1. **AND (Conjunción \- Producto Lógico):** La salida es 1 solo si ambas entradas son 1 ($A \cdot B$).  
2. **OR (Disyunción \- Suma Lógica):** La salida es 1 si al menos una de las entradas es 1 ($A + B$).  
3. **NOT (Negación \- Inversor):** Invierte el valor de entrada; si entra 0, sale 1 ($\neg A$).  
4. **XOR (O Exclusivo):** La salida es 1 solo si las entradas son diferentes ($A \oplus B$).

El software incluye validaciones para asegurar que el usuario solo ingrese datos dentro del dominio binario y utiliza representaciones gráficas (Arte ASCII) para facilitar la comprensión visual del circuito.
